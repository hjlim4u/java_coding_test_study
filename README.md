### github 올리는 방법
1. git pull origin main
2. git branch [브랜치명]
3. git add . 
4. git commit -m "message내용"
5. git push origin [생성한 브랜치명]
6. merge request

### 문제풀이 후 github 업로드
1. 백준 허브 크롬 확장 도구 설치
    - [백준허브] : https://github.com/BaekjoonHub/BaekjoonHub
2. 문제 풀이 후 제출
3. 파일명 수정 (문제이름 -> 문제이름_본인이름)
